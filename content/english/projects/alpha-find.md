#### 1. 팀장
- **9기 최인열**

#### 2. 멤버
- 심민성, 김진호, 이재성, 박영선, 최진우, 이문기

#### 3. 활동 계획
- **방식:** 디스코드 온라인 모각코 (캠 켜고 각자 코딩)
- **목표:** 시장에 숨겨진 알파를 찾아 공유하고 실현 가능한 인사이트 발견

#### 4. 활동 내용
- 흔한 알파가 실제로 동작하는지 검증
- 동작 시 실전 자동매매 구현 (빗썸, 업비트, 증권사 API 활용)
- 매매일지 작성 및 월간 수익률 공유
- (코딩 초보자도 참여 가능)

#### 5. 학습 자료 및 데이터
- 위키독스, 팀장 제공 소스
- 자체 수집 데이터 및 증권사 API
